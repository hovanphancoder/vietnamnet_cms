# API Endpoints Documentation

## Overview
API endpoints for the system.

## Authentication
- POST /api/auth/login
- POST /api/auth/logout
- POST /api/auth/refresh

## Users
- GET /api/users
- POST /api/users
- GET /api/users/{id}
- PUT /api/users/{id}
- DELETE /api/users/{id}

## Files
- GET /api/files
- POST /api/files
- GET /api/files/{id}
- PUT /api/files/{id}
- DELETE /api/files/{id}
